# TP-Conculandia

Simulación concurrente de la oficina de inmigraciones de **Conculandia** (_Técnicas de Programación Concurrente I - FIUBA_)  

### Compilar

```bash
$ make
```

### Ejecutar

- Modo normal
```bash
$ make run [V=v | S=s | D=-d]

  (V, S, D son opcionales y por default tiene valores V=4, S=2, D="")
```

- Modo debug
```bash
$ make run-debug
```

### Formato

```bash
$ make format
```
