# TP-Conculandia

Simulación concurrente de la oficina de inmigraciones de **Conculandia** (Técnicas de Programación Concurrente I)  

### Compilar

```bash
$ make
```

### Ejecutar

- Modo normal
```bash
$ make run
```

- Modo debug
```bash
$ make run-debug
```

### Formato

```bash
$ make format
```
